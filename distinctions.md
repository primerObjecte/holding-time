# Key Distinctions

## Gap vs Object

| Gap | Object |
|-----|--------|
| Primary | Secondary |
| Holds tension | Collapses tension |
| Does not flow | Exists in time |
| Admits, separates, self-durates | Is admitted, separated, durated |

## Holding vs Physical Time

| Holding | Physical Time |
|---------|---------------|
| No direction | Direction (past → future) |
| No accumulation | Accumulates (entropy) |
| Exists with any difference | Exists between events |

## Holding vs Bion's Negative Capability

| Holding | Negative Capability |
|---------|---------------------|
| Ontological | Psychological |
| Pre-subjective | Human capacity |
| Time that does not flow | Tolerance of uncertainty in time |
